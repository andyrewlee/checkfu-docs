## The Student Journey

1. **Get invited** - Student receives email to join your workspace
2. **Join** - They create an account or sign in
3. **Browse courses** - See what's available
4. **Start a lesson** - Work through content
5. **Complete work** - Fill in worksheets, answer questions
6. **Take quizzes** - Test their understanding
7. **Get feedback** - See grades and comments

## What Students See

### Dashboard
After logging in:
- Courses they're enrolled in
- Current lesson / what's next
- Recent activity
- Progress summary

### Course View
Inside a course:
- Course description
- List of lessons (with progress indicators)
- Locked lessons (if sequential mode)
- AI tutor access

### Lesson View
Inside a lesson:
- Video or text content
- Worksheets to complete
- Quiz (if attached)
- AI tutor chat

## What Students Can Do

| Action | Description |
|--------|-------------|
| **Complete worksheets** | Fill in answers on interactive content |
| **Submit work** | Send completed work for grading |
| **Take quizzes** | Answer adaptive quiz questions |
| **Ask AI tutor** | Get help without giving away answers |
| **View feedback** | See grades and teacher comments |
| **Track progress** | See how far they've come |

## Requirements

To access Checkfu:
- Email address (or school SSO)
- Invitation to your workspace
- Web browser (Chrome, Safari, Firefox, Edge)
