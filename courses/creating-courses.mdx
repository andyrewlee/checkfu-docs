This guide walks through creating a course step by step.

## Create a Course

1. Click **Courses** in the sidebar
2. Click **New Course**
3. Enter a name
4. Configure settings (see below)

## Course Settings

### Basic Info
- **Name**: What students will see
- **Description**: Optional overview

### Progress Mode
- **Sequential**: Students must complete lessons in order
- **Flexible**: Students can do lessons in any order

### Default Agent
Pick an AI tutor that will help students throughout this course. (You can override this for specific lessons.)

## Building Your Course

### Adding Sections

Sections help organize lessons into groups:

1. Click **Add Section**
2. Enter a name (e.g., "Unit 1: Introduction")
3. Drag sections to reorder them

Sections are optionalâ€”you can add lessons directly to the course without sections.

### Adding Lessons

1. Click **Add Lesson** (within a section or at the root)
2. Enter a lesson name
3. Configure the lesson content (see [Lessons](/courses/lessons))

### Reordering

- Drag sections to change their order
- Drag lessons within or between sections
- Drag a lesson outside all sections to make it standalone

## AI-Generated Course Outlines

Want AI to help plan your course?

1. Click **Generate Course**
2. Describe what you want:
   ```
   "A 4-week course on multiplication for 3rd graders,
   starting with basic facts and progressing to
   multi-digit multiplication"
   ```
3. Review the generated outline
4. Edit as needed
5. Generate content for individual lessons

## Publishing

When your course is ready:

1. Review all lessons and content
2. Click **Publish** in course settings
3. Status changes to **Published**
4. Students can now see the course

<Warning>
Students can only see Published courses. Keep it in Draft while you're building.
</Warning>

## After Publishing

Once published, you can track:
- Student enrollment
- Lesson completion rates
- Quiz scores
- Time spent per lesson
